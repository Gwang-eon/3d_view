옹벽 3D 뷰어 - 로컬 실행 도우미

이 파일은 로컬에서 옹벽 3D 뷰어를 실행하기 위한 도우미 페이지입니다.

중요: 브라우저의 보안 제한으로 인해 로컬 파일에서 직접 GLTF 모델을 로드할 수 없습니다. 간단한 로컬 웹 서버를 사용하여 실행해야 합니다.

실행 방법:

  1. Python 사용:
    Python이 설치되어 있다면, 다음 명령어로 간단한 HTTP 서버를 실행할 수 있습니다:
    python -m http.server (Python 3)
    python -m SimpleHTTPServer (Python 2)
  2. Node.js 사용:
    Node.js가 설치되어 있다면, 다음 명령어로 http-server를 설치하고 실행할 수 있습니다:
    npx http-server
  3. Visual Studio Code 사용:
    VS Code의 Live Server 확장 프로그램을 설치하고 사용할 수 있습니다.

서버가 실행되면, 아래 버튼을 클릭하여 옹벽 3D 뷰어를 실행하세요:

옹벽 3D 뷰어 실행

문제 해결:

  • 모델이 로드되지 않는 경우, 콘솔(F12)에서 오류 메시지를 확인하세요.
  • 파일 경로 문제가 있다면, js/config.js 파일에서 modelsPath 설정을 확인하세요.
  • 서버가 올바른 디렉토리에서 실행되었는지 확인하세요 (wall_viewer 폴더).
